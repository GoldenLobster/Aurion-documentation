Welcome to Aurion

Aurion is a modern open-source music player with offline downloads, playlists, and an extensible plugin-friendly architecture.

Download GitHub

Why Aurion?

Play Everywhere

Seamless playback with smart queues and cross-platform clients.

Download Music

Save music for offline listening where legally permitted; manage download quality and lifecycle.

Privacy-First

No tracking, local-first designs, and transparent open-source code.

Quickstart

# Clone
git clone https://github.com/GoldenLobster/Aurion.git
cd Aurion

# Example: run local dev build
npm install
npm run start
