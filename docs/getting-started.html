Getting Started

System Requirements

  • Desktop: Windows / macOS / Linux
  • Mobile: Android (iOS support TBD)
  • Node.js & npm (for building docs or development): Node >= 16

Install from source (development)

git clone https://github.com/GoldenLobster/Aurion.git
cd Aurion
npm install
npm run start

Configuration

Config example file (put into platform-specific config path):

{
  "downloadPath": "~/Music/Aurion",
  "quality": "320",
  "autoDownloadOnWifi": true
}
