Roadmap

My mind is all over the place. Whatever I come up with I will try to note down and put it here, but chances are that I might release new versions without really explaning whats new.

Planned Features

  • Smooth animation for the text(artist name and song name)
  • new icons(volume and folder icons)
  • Lyrics integration
  • Making the settings an actual full tab instead of a little window
  • while in another tab(like the settings tab for example) there will be a small control bar at the bottom of the screen
  • Integrate the downloader into a tab instead of seperate window
  • *once all the above is done the projects gui framework will be entirely pyqt5*
  • Another addition to the downloader: having the ability to transfer playlists from another music service to youtube for easy downloading
  • Making the downloader look more like the youtube music site(having recommendations, showing thumbnails, being able to pass cookies to see personalized recommendations, etc)
