Features

  • High-quality playback of almost any audio format(depends on system codecs)
  • Music downloader powered by yt-dlp and ytmusicapi for searching and downloading songs.
  • Playlists, queue management, repeat/shuffle
  • Crossfade between songs.
  • Search by song name or artist name.
  • Clean intuitive UI.
